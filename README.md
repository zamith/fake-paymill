# FakePaymill

A Paymill fake so that you can avoid hitting Paymill's servers in tests

## Installation

Add this line to your application's Gemfile:

    gem 'fake_paymill'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fake_paymill
